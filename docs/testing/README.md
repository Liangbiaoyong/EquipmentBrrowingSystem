## 测试策略概览

| 类型 | 工具 | 执行时机 |
|------|------|----------|
| 单元测试 | JUnit5 + Mockito | 每次提交前 |
| 接口测试 | Postman + Newman | 提测前 |
| 集成测试 | Testcontainers | 模块完成时 |
| 性能测试 | JMeter（30 并发） | 核心模块完成后 |
| 安全测试 | OWASP ZAP + 手动 | 上线前 |
| 兼容性 | 手动测试 | 前端完成 80% 时 |

详细测试用例、CI 配置和验收场景请阅读 [测试编排.md](测试编排.md)。
